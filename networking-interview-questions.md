# Networking Interview Questions

## 1. What is a Computer Network?

A computer network is a collection of interconnected devices that communicate and share resources.

---

## 2. What is the OSI Model?

The OSI (Open Systems Interconnection) Model is a 7-layer framework used to understand network communication.

Layers:

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

---

## 3. What is the TCP/IP Model?

TCP/IP is the protocol suite used on the Internet.

Layers:

* Application
* Transport
* Internet
* Network Access

---

## 4. What is an IP Address?

An IP address is a unique identifier assigned to a device on a network.

Example:
192.168.1.1

---

## 5. Difference Between IPv4 and IPv6?

| IPv4                 | IPv6                 |
| -------------------- | -------------------- |
| 32-bit               | 128-bit              |
| Limited addresses    | Huge address space   |
| Example: 192.168.1.1 | Example: 2001:db8::1 |

---

## 6. What is a MAC Address?

A MAC (Media Access Control) Address is a unique hardware address assigned to a network interface card.

Example:
00:1A:2B:3C:4D:5E

---

## 7. What is DNS?

DNS (Domain Name System) translates domain names into IP addresses.

Example:
google.com → 142.250.x.x

---

## 8. What is DHCP?

DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses to devices.

DHCP Process:

* Discover
* Offer
* Request
* Acknowledge

(DORA)

---

## 9. What is Subnetting?

Subnetting is the process of dividing a network into smaller logical networks.

Benefits:

* Better performance
* Improved security
* Efficient IP utilization

---

## 10. Difference Between TCP and UDP?

| TCP                 | UDP               |
| ------------------- | ----------------- |
| Reliable            | Unreliable        |
| Connection-Oriented | Connectionless    |
| Slower              | Faster            |
| Error Recovery      | No Error Recovery |

---

## 11. What is Routing?

Routing is the process of forwarding packets from one network to another.

Routing Protocols:

* RIP
* OSPF
* EIGRP
* BGP

---

## 12. Difference Between Hub and Switch?

### Hub

* Broadcasts data to all devices
* Less secure
* Works on Physical Layer

### Switch

* Sends data only to intended device
* More secure
* Works on Data Link Layer

---

## 13. Difference Between Switch and Router?

| Switch                  | Router                      |
| ----------------------- | --------------------------- |
| Connects devices in LAN | Connects different networks |
| Uses MAC Address        | Uses IP Address             |
| Layer 2                 | Layer 3                     |

---

## 14. What is a Firewall?

A firewall is a security device/software that filters incoming and outgoing network traffic.

Functions:

* Block unauthorized access
* Protect network resources

---

## 15. What is VPN?

VPN (Virtual Private Network) creates a secure encrypted connection over the Internet.

Benefits:

* Privacy
* Security
* Remote Access

---

## 16. What is NAT?

NAT (Network Address Translation) converts private IP addresses into public IP addresses.

Benefits:

* Conserves IPv4 addresses
* Improves security

---

## 17. What is HTTP?

HTTP (HyperText Transfer Protocol) is used for communication between web browsers and web servers.

Default Port:
80

---

## 18. What is HTTPS?

HTTPS is the secure version of HTTP.

Default Port:
443

Uses:

* SSL/TLS Encryption
* Secure Communication

---

## 19. What is Ping?

Ping is a network utility used to test connectivity between devices.

Command:

ping google.com

---

## 20. What is Traceroute?

Traceroute shows the path packets take to reach a destination.

Commands:

Windows:

tracert google.com

Linux:

traceroute google.com
