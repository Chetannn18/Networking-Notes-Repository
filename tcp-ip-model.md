# TCP/IP Model

TCP/IP is the protocol suite used on the Internet.

## Layers

| Layer | Protocols |
|---------|---------|
| Application | HTTP, FTP, DNS |
| Transport | TCP, UDP |
| Internet | IP, ICMP |
| Network Access | Ethernet, Wi-Fi |

## TCP Features

- Reliable
- Connection Oriented
- Error Checking

## UDP Features

- Fast
- Connectionless
- No Error Recovery
